**JWT and JWS (JOSE)**
IETF specifications and best-practice guidance.

- RFC 7519, "JSON Web Token (JWT)". Core JWT specification. https://www.rfc-editor.org/info/rfc7519
- RFC 7515, "JSON Web Signature (JWS)". Defines signing and compact/JSON serializations. https://www.rfc-editor.org/rfc/rfc7515
- RFC 7517, "JSON Web Key (JWK)". Defines key representation used with JOSE. https://www.rfc-editor.org/rfc/rfc7517
- RFC 7518, "JSON Web Algorithms (JWA)". Defines algorithms and registries for JWS/JWE/JWK. https://www.rfc-editor.org/info/rfc7518
- RFC 8725, "JSON Web Token Best Current Practices". Updates RFC 7519 with security guidance. https://www.rfc-editor.org/info/rfc8725
- RFC 7797, "JWS Unencoded Payload Option". Adds the "b64" header parameter and notes JWTs must not use unencoded payloads. https://www.rfc-editor.org/info/rfc7797
